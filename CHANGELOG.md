# Changelog

Player-facing changes to Hand of Fate Access, newest first. An entry describes what a player will hear or notice: a new or changed announcement or sound, new controls, a fixed silence. Internal work (refactors, tests, tooling, docs) does not get entries.

New changes go under Unreleased as they are made; the release skill rolls that section into a version heading when a release is cut.

## Unreleased

- Zoomed-card decisions and the whole-set card choice now name the confirm and cancel inputs after each option ("Yes: Return, No: Tab" on keyboard, "Yes: A, No: B" on controller); the lookup always failed, so the hint carried only the bare labels. Input names read only when there are two options to tell apart; a lone option stays bare.
- The sound glossary option in the pause menu now switches language with the game; it used to keep the language it was first created in until a scene change.
